# TVBox:嗅探播放器
## 配置教程
1. 通过USB或ADB安装[TVBox空壳](#空壳下载)
2. 打开空壳APP进入设置
3. 在点播输入[接口地址](#接口地址)
    - 在线地址:直接输入或局域网内扫码输入
    - 本地包:解压到根目录下TVBox文件夹并进入APP允许文件访问权限
      - 影视输入`file://TVBox/api.json`
      - TVBox输入`clan://localhost/TVBox/api.json`
4. 点击确定完成输入并返回到主界面
